## 安装依赖

```bash
# pnpm
pnpm install
```

## 项目启动

启动在 `http://localhost: 8888`

```bash
pnpm dev
```

## 打包构建

```bash
pnpm  build
```

## 项目预览
```bash
pnpm preview
```
